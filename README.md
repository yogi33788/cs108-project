The project is about creating a secure 2 player GUI based MINI GAME HUB
Two authenticated players can select a game from the menu, play through a graphical interface, and save their results to a persistent leaderboard.

The project consists of 3 games namely 
1.Tic Tac Toe
2.Othello
3.Connect4
The specific rules are as follows
for Tictactoe : https://tictactoefree.com/rules
for Othello :https://www.mathsisfun.com/games/reversi.html
for Connect4 :https://www.mathsisfun.com/games/connect4.html

The Bash login and a hash method to secure the passwords is being used for profesionality
Game results , using matplotlib and numpy arrays for winning conditions. Matlpotlib for visualization of the results of games on terminal integrated with multiple
options for display

the game is all about fun and a refreshing feel to the players
